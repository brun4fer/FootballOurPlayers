# Manual de Utilização  
## Plataforma de Análise de Futebol

## 1. Introdução
Esta plataforma foi criada para apoiar treinadores e analistas na avaliação do desempenho de jogadores e da equipa ao longo das jornadas de uma competição.

Com ela, pode:
- registar informação de jogos e jogadores;
- acompanhar métricas individuais;
- analisar o rendimento coletivo da equipa;
- comparar evolução entre jornadas.

O objetivo é transformar dados de jogo em informação útil para apoiar decisões técnicas.

## 2. Visão Geral da Plataforma
Em seguida encontra as principais áreas do sistema e a sua função:

- **Overview**: página inicial com acesso rápido às áreas principais.
- **Admin Home**: ponto central de gestão e configuração.
- **Seasons**: criação e gestão de épocas desportivas.
- **Competitions**: criação e gestão de competições em cada época.
- **Teams**: registo e gestão de equipas.
- **Players**: registo e gestão de jogadores.
- **Matches**: criação dos jogos/jornadas.
- **Stats Entry**: inserção das estatísticas por jogador e por jogo.
- **Dashboard de Jogadores**: análise detalhada de desempenho individual.
- **Dashboard de Equipas**: análise global da equipa por competição e jornadas.

## 3. Inserção de Dados
Para garantir análises corretas, a introdução de dados deve seguir uma ordem lógica.

### 3.1 Adicionar Equipas
1. Entre em **Admin Home**.
2. Abra **Teams**.
3. Clique em criar/adicionar equipa.
4. Preencha os dados pedidos.
5. Grave.

### 3.2 Adicionar Jogadores
1. Entre em **Players**.
2. Clique em criar/adicionar jogador.
3. Selecione a equipa do jogador.
4. Preencha os restantes dados.
5. Grave.

### 3.3 Criar Jornadas / Jogos
1. Entre em **Matches**.
2. Clique em criar jogo.
3. Selecione:
   - competição;
   - jornada;
   - adversário;
   - data;
   - casa/fora.
4. Grave.

### 3.4 Inserir Estatísticas dos Jogadores
1. Entre em **Stats Entry**.
2. Escolha primeiro a **competição**.
3. Escolha o **jogo/jornada**.
4. Escolha o **jogador**.
5. Introduza os valores estatísticos.
6. Grave.
7. Repita para todos os jogadores utilizados no jogo.

## 4. Dashboard de Jogadores
Esta área permite analisar o desempenho de um jogador em detalhe.

### Filtros disponíveis
- **Competição**
- **Jogador**
- **Jornada**
- **Selecionar múltiplas jornadas**
- **Selecionar múltiplos jogadores**

### Como usar
1. Selecione a competição.
2. Escolha um jogador para análise individual.
3. Use uma jornada específica ou várias jornadas para ver evolução.
4. Aplique os filtros.

### Secções principais
- **Ações Percentuais**: mostra eficácia (sucesso/insucesso e percentagem).
- **Ações Numéricas**: mostra totais e valores por 90 minutos.
- **Gráficos de Evolução**: mostra a variação de métricas ao longo das jornadas.

Estas secções ajudam a perceber consistência, momentos de melhoria e impacto do jogador.

## 5. Dashboard de Equipas
Esta área mostra o desempenho global da equipa.

### Filtros disponíveis
- **Competição**
- **Jornada**
- **Múltiplas jornadas**

### Lógica de análise
- Competição apenas: visão total da competição.
- Competição + 1 jornada: análise de um jogo específico.
- Competição + múltiplas jornadas: análise da evolução entre jogos.

### Secções principais
- **Gráfico de Ações Ofensivas**
- **Ações Percentuais**
- **Ações Numéricas**
- **Gráficos Evolutivos**

As estatísticas da equipa são calculadas automaticamente com base nas estatísticas dos jogadores inseridas em cada jogo.

## 6. Relatório Público
A funcionalidade **Gerar Relatório Público** permite criar um link partilhável com a análise atual.

### Como gerar
1. Abra o **Dashboard de Jogadores**.
2. Aplique os filtros que pretende (competição, jogador, jornadas, etc.).
3. No final da página, clique em **Gerar Relatório Público**.

### O que o relatório inclui
- os filtros aplicados no momento da geração;
- visão de leitura (sem edição);
- tabelas e gráficos da análise correspondente.

### Como partilhar
1. Após gerar, é aberto um link único do relatório.
2. Partilhe esse link com equipa técnica, direção ou outros analistas.

## 7. Boas Práticas de Utilização
- Registe os dados logo após cada jogo.
- Confirme os valores antes de gravar.
- Use sempre os mesmos critérios de registo para manter consistência.
- Compare várias jornadas para identificar tendências.
- Analise jogador e equipa em conjunto para uma visão mais completa.
- Gere relatórios públicos quando precisar de partilhar análise com terceiros.

---
Se seguir este fluxo de trabalho, terá uma base de análise fiável e útil para apoiar decisões técnicas ao longo da época.
